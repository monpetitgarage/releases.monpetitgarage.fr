# MonPetitGarage - Releases

Binaires officiels de l'application desktop [MonPetitGarage](https://www.monpetitgarage.fr).

## Téléchargement

Rendez-vous sur la [page de téléchargement](https://www.monpetitgarage.fr/telecharger) ou directement dans les
[Releases](https://github.com/monpetitgarage/releases.monpetitgarage.fr/releases/latest).

| Plateforme | Fichier |
|------------|---------|
| **macOS Apple Silicon** | `.dmg` (aarch64) |
| **macOS Intel** | `.dmg` (x64) |
| **Windows** | `.exe` ou `.msi` |
| **Linux** | `.AppImage` ou `.deb` |

## Mise à jour automatique

L'application desktop intègre un système de mise à jour automatique. Lorsqu'une nouvelle version est disponible, une notification
apparaît et vous pouvez installer la mise à jour en un clic.

Le fichier `latest.json` est utilisé par l'auto-updater pour vérifier les nouvelles versions.

## Vérification des signatures

Chaque binaire est accompagné d'un fichier `.sig` contenant sa signature cryptographique, garantissant l'intégrité et l'authenticité
des fichiers.
